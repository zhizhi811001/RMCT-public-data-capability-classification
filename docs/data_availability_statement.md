\# Data Availability Statement



This repository provides derived research data and reproducibility materials for the study \*\*A public-data-driven, taxonomy-enabled capability classification framework for regional manufacturing visibility and partner discovery\*\*.



\## Data Sources



The study uses public-data sources only. These include:



1\. a Companies House-derived manufacturer population frame for the West Midlands manufacturing ecosystem; and  

2\. publicly accessible manufacturer website pages collected for capability evidence extraction.



The website data were used to identify public evidence of manufacturing capabilities, including processes, materials, certifications, sectors served, engineering support, digital capability, commercial readiness and sustainability-related practices.



\## Data Shared in This Repository



To support verification and reproducibility, this repository shares derived data rather than full website text. Shared files may include:



\- company identifiers;

\- website URLs where available;

\- crawl dates;

\- document hashes;

\- extracted capability evidence spans;

\- RMCT preferred labels;

\- alternative labels;

\- broader RMCT dimensions;

\- narrower capability categories;

\- anonymised company-level capability profiles;

\- benchmark briefs;

\- aggregate benchmark results;

\- expert-validation templates and summary metrics, where applicable.



These files are intended to allow readers to inspect how public evidence was converted into RMCT capability labels and company-level capability profiles.



\## Data Not Shared



The repository does not redistribute:



\- full raw website HTML;

\- full cleaned website text;

\- complete scraped page archives;

\- personal contact details;

\- email addresses;

\- phone numbers;

\- contact-form content;

\- any information not required for reproducing the capability classification results.



Full website text is not shared because manufacturer websites may be subject to copyright, website terms of use, and later modification or removal by the website owner.



\## Verification Approach



Instead of full text redistribution, this repository provides:



\- short capability evidence spans;

\- source URLs where available;

\- crawl dates;

\- document-level hashes;

\- assigned RMCT labels;

\- aggregate analysis outputs.



This approach allows the analysis to be checked while limiting redistribution of copyrighted or potentially sensitive website content.



\## Access to Additional Data



Additional derived data may be made available on reasonable request where this is consistent with copyright, privacy, ethical and data-governance constraints. Raw or full cleaned website texts will not normally be shared.



\## Data Citation



If using the derived datasets or code from this repository, please cite the associated manuscript and this repository. A formal citation will be added after publication.
