# Figures

This folder contains publication-oriented figures used to communicate the RMCT workflow and validation results. TIFF files are not included in the GitHub version to keep the repository lightweight; PNG, SVG and PDF versions are provided where available.

## Figure index

| Figure file prefix | Description | Manuscript role |
|---|---|---|
| `figure1_from_sic_to_capability_visibility` | From SIC-based visibility to capability-level visibility | Motivates the move from industrial activity codes to capability-level supplier discovery |
| `figure2_rmct_decision_aid_model` | RMCT as a public-data-driven decision-aid model | Shows the decision-support pipeline from public data to supplier search |
| `figure3_literature_derived_dimension_construction` | Literature-derived dimension construction | Shows how literature concepts support candidate capability dimensions |
| `figure4_capability_level_visibility_enabled_by_rmct` | Capability-level visibility enabled by RMCT | Illustrates firm-level capability profiles and capability-based visibility |
| `figure5_literature_website_cluster_crosscheck` | Literature dimensions versus website-derived clusters cross-check | Shows empirical cross-check between website clusters and RMCT dimensions |
| `figure6_rmct_controlled_vocabulary_company_profiles` | Controlled vocabulary and company profile construction | Shows how heterogeneous website terms become preferred labels and company profiles |
| `figure6_benchmarking_results_against_baselines` | Benchmarking results against baselines | Summarises comparative validation results |
| `figure6_population_coverage_benchmark` | Population coverage benchmark | Compares reach across classification/search systems |
| `figure6_capability_specificity_benchmark` | Capability specificity benchmark | Compares descriptor richness and structured capability fields |
| `figure6_balanced_three_baseline_search_noise_comparison` | Search noise reduction across baselines | Compares RMCT against SIC, keyword-only and single-dimension baselines |

## Reuse note

These figures are shared for scholarly review and reproducibility of the associated manuscript. Please cite the manuscript or repository if reusing them.
